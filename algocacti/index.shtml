Algo-cacti

Photos by Victoria Holt.

We created the Algo-cacti system in order to easily set up interactive projection mappings for organic objects like cacti. We initially developed it after being invited to create an installation for a Seattle Volunteer Park Conservatory fundraiser. The software can be configured for different styles of interaction (for example, an arcade track ball that modulates the wiggliness of the cacti).
